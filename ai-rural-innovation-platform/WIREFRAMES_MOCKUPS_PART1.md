## Color Scheme

- **Primary**: #0066CC (Blue) - Trust, technology
- **Secondary**: #4CAF50 (Green) - Growth, agriculture
- **Accent**: #FF9800 (Orange) - Action, alerts
- **Success**: #4CAF50 (Green) - Positive status
- **Warning**: #FFC107 (Yellow) - Caution
- **Error**: #F44336 (Red) - Issues
- **Background**: #FFFFFF (White) - Clean
- **Text**: #333333 (Dark Gray) - Readable

---

## Typography

- **Headers**: Bold, 24-28px
- **Subheaders**: Bold, 18-20px
- **Body**: Regular, 14-16px
- **Small**: Regular, 12-14px
- **Font Family**: Inter, Roboto, or system font

---

## Component Library

### Buttons
- Primary Button: Blue background, white text
- Secondary Button: White background, blue border
- Danger Button: Red background, white text
- Disabled Button: Gray background, gray text

### Input Fields
- Text Input: Border, padding, focus state
- Dropdown: Arrow icon, options list
- Checkbox: Square, checkmark
- Radio: Circle, filled state

### Cards
- Elevation: Subtle shadow
- Padding: 16px
- Border Radius: 8px
- Hover: Slight elevation increase

### Icons
- Size: 24px (standard), 32px (large), 16px (small)
- Color: Match text color or accent
- Stroke: 2px for consistency
